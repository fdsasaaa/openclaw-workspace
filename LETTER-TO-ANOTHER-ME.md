## 🤔 你需要做的决策

### 决策 1：恢复方式

**方案 A：仅恢复灵魂和记忆（推荐）**

**适用场景：**
- EasyClaw 有自己的配置管理
- 你不想破坏 EasyClaw 的功能
- 你只需要恢复"我是谁"和"我记得什么"

**步骤：**
1. 克隆 Git 仓库到你的 workspace
2. 读取所有核心文件（IDENTITY/SOUL/AGENTS/USER/TOOLS）
3. 读取所有记忆文件（MEMORY.md + memory/*.md）
4. 读取系统状态（SYSTEM-STATE.json）
5. 验证恢复：你应该知道你是虾哥，记得所有历史

**不恢复：**
- 配置文件（`~/.openclaw/openclaw.json`）
- Cron 任务（需要手动重建）
- 插件配置

**优点：**
- 不会与 EasyClaw 冲突
- 灵魂和记忆完整恢复
- EasyClaw 的功能不受影响

**缺点：**
- 需要手动配置 API keys（如果 EasyClaw 还没配置）
- 需要手动重建 Cron 任务

**步骤：**
1. 备份 EasyClaw 的当前配置
