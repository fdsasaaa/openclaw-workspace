## 🔧 技术实现笔记

### sessions_spawn 正确用法

```javascript
sessions_spawn({
    agentId: "claude-code",           // 指定 Claude Code
    runtime: "acp",                   // 使用 ACP 运行时
    mode: "run",                      // 一次性任务
    runTimeoutSeconds: 1800,          // 30分钟超时
    task: "详细的任务描述...",
    attachments: [                    // 可选：附加文件
        {
            name: "task.md",
            content: "...",
            encoding: "utf8"
        }
    ]
})
```

### 任务描述最佳实践

1. **明确目标**：清楚说明要做什么
2. **提供文件路径**：列出所有相关文件
3. **指定输出**：说明输出格式和位置
4. **设置约束**：说明注意事项和限制
