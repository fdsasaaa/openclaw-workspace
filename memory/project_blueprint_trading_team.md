## 一、核心资源

### 1.1 现有策略A：半自动箱体交易
- **类型**: 半自动EA (MT4)
- **机制**: 人工画箱体 → EA自动交易
- **状态**: 实盘运行，回撤和盈利已验证
- **平台**: MT4

### 1.2 现有策略B：全自动箱体交易
- **类型**: 全自动策略 (MT5)
- **机制**: 策略自动识别并画箱体 → 自动交易
- **回测**: 近年数据 + 24年长期数据均稳定盈利
- **实盘**: 微信信号 → 桥接 → MT4，已产生盈利
- **状态**: 已验证，可商用

## 六、中枢反馈总结

### 6.1 可行性评估

| 维度 | 评分 | 说明 |
|------|:----:|------|
| 技术可行性 | 70% | 策略已验证，AI部分有挑战 |
| 商业可行性 | 60% | 市场竞争激烈，需差异化 |
| 资源需求 | 高 | 需要持续投入时间和资金 |
| 风险等级 | 中高 | 监管、技术、市场风险并存 |

### 6.2 核心建议

1. **先验证，再建设**
   - 不要一开始就建设完整AI团队
   - 先手动服务10-20个客户，验证付费意愿

2. **从简单开始**
   - 先做落地页 + 支付 + 手动服务
   - 再逐步自动化

3. **合规优先**
   - 交易信号服务涉及金融监管
   - 先咨询法律意见

4. **保留现金牛**
   - 不要all in这个项目
   - 保持现有盈利策略的运行
