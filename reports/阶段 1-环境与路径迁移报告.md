## 四、系统目录映射说明

### 4.1 目录分类

| 类别 | 路径 | 可迁移 | 说明 |
|-----|------|-------|------|
| 🔴 **系统核心** | `C:\...\node_modules\openclaw` | ❌ 否 | 程序本体、官方技能，不可移动 |
| 🟡 **配置运行时** | `C:\Users\Administrator\.openclaw` | ⚠️ 部分 | 主配置不可动，工作区可迁移 |
| 🟢 **用户数据** | `E:\OpenClaw_Workspace\*` | ✅ 是 | 全部在 E 盘，可自由组织 |

### 4.2 关键映射关系

| 逻辑用途 | 当前实际路径 | 目标路径 | 迁移状态 |
|---------|-------------|---------|---------|
| 主配置文件 | `C:\Users\Administrator\.openclaw\openclaw.json` | 同左 | 🔒 保留 |
| 工作区 | `C:\Users\Administrator\.openclaw\workspace` | `E:\OpenClaw_Workspace\workspace` | ⏸️ 待配置 |
| 记忆文件 | `C:\Users\Administrator\.openclaw\workspace\memory\` | `E:\OpenClaw_Workspace\memory\` | ⏸️ 待配置 |
| 系统日志 | `C:\Users\Administrator\.openclaw\logs\` | 同左 | 🔒 保留 |
| 用户日志 | - | `E:\OpenClaw_Workspace\logs\` | ✅ 就绪 |
| 项目代码 | - | `E:\OpenClaw_Workspace\projects\` | ✅ 就绪 |
| 自定义技能 | - | `E:\OpenClaw_Workspace\skills\` | ✅ 就绪 |

### 4.3 为什么有些目录不能迁移

| 目录 | 原因 |
|-----|------|
| `node_modules\openclaw` | npm 全局安装包，移动后命令无法找到 |
| `.openclaw\openclaw.json` | 主配置文件，多处硬编码引用 |
| `.openclaw\gateway.cmd` | 网关启动脚本，任务计划程序引用 |
| `.openclaw\logs` | 系统日志，程序硬编码路径 |
| `.openclaw\agents` | Agent 运行时数据，程序硬编码路径 |
