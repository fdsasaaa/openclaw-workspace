# 结果展示布局设计

## 目标

设计清晰、专业、易懂的交易结果展示方式，让观众快速理解策略表现。

---

## 布局方案：全屏MT5 + 数据覆盖层

### 整体布局

```
┌─────────────────────────────────────────────────────────┐
│  [日期] Day 1 - March 7, 2026        [盈亏] +$XXX  ↑   │  ← 顶部信息栏
├─────────────────────────────────────────────────────────┤
│                                                         │
│                                                         │
│                  MT5 图表区域                           │
│                  (全屏显示)                             │
│                                                         │
│                                                         │
│                                                         │
├─────────────────────────────────────────────────────────┤
│  [交易1] Entry: 1.3900 → Exit: 1.3950 | +50 pips      │  ← 底部交易信息
└─────────────────────────────────────────────────────────┘
```

## 过渡动画

### 交易切换动画

**效果1：淡入淡出**
- 当前交易淡出（0.5秒）
- 新交易淡入（0.5秒）

**效果2：滑动切换**
- 当前交易向左滑出
- 新交易从右滑入

**推荐：效果1（淡入淡出）**
- 更平滑
- 不分散注意力

---

### 数据更新动画

**效果：数字滚动**
- 盈亏数字变化时

## 配音同步

### 配音与画面同步

**原则：**
- 配音提到交易时，画面显示该交易
- 配音提到数据时，数据高亮显示
- 配音停顿时，画面保持静止

**示例时间轴：**
```
0:30 - "Let's look at the first trade..."
       → 显示 Trade #1 标注

0:45 - "Entry at 1.3900..."
       → 入场标注闪烁

1:00 - "Exit at 1.3950, profit 50 pips"
       → 出场标注闪烁，盈亏数字滚动
```

## 技术实现

### FFmpeg 命令

**1. 添加顶部信息栏**
```bash
ffmpeg -i input.mp4 -vf "drawbox=x=0:y=0:w=iw:h=80:color=black@0.7:t=fill,
drawtext=text='Day 1 - March 7, 2026':x=20:y=20:fontsize=24:fontcolor=white" output.mp4
```

**2. 添加底部交易信息**
```bash
ffmpeg -i input.mp4 -vf "drawbox=x=0:y=ih-60:w=iw:h=60:color=black@0.8:t=fill,
drawtext=text='Trade #1 | EURUSD H1 | +50 pips':x=20:y=ih-40:fontsize=18:fontcolor=white" output.mp4
```

**3. 添加箭头标注（使用图片覆盖）**
```bash
ffmpeg -i input.mp4 -i arrow.png -filter_complex "[0:v][1:v]overlay=x=100:y=200" output.mp4
```
