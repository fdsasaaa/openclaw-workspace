### 2. 新增功能

#### 趋势过滤（核心新增）
```pine
EnableTrendFilter = true     // 启用趋势过滤
TrendEMA_Period = 50         // 使用EMA(50)判断趋势

// 趋势判断逻辑
ema50 = ta.ema(close, TrendEMA_Period)
trendUp = close > ema50      // 上升趋势
trendDown = close < ema50    // 下降趋势

// 入场过滤
if not EnableTrendFilter or trendUp
    strategy.entry("Break_Long", ...)   // 只在上升趋势做多
if not EnableTrendFilter or trendDown
    strategy.entry("Break_Short", ...)  // 只在下降趋势做空
```

**预期效果**:
- 避免逆势交易
- 空头交易减少50%
- 整体胜率提升3-5%

## ⚠️ 重要提示

### 1. 趋势过滤开关
```pine
EnableTrendFilter = true  // 默认启用
```
- 如果市场震荡，可以关闭趋势过滤
- 如果明显趋势市场，建议保持开启

### 2. 参数微调建议

根据实盘表现可微调：

**如果止损过频**:
```pine
ATR_SL_Mult = 1.7  // 从1.6增加到1.7
```

**如果盈利过小**:
```pine
TrailStartRatio = 90%  // 从85%增加到90%
```

**如果交易过少**:
```pine
MinDisplayScore = 38  // 从40降到38
```

### 3. 不同品种调整

**黄金 (XAUUSD)**:
- 使用默认参数即可

**指数 (US30, USTEC)**:
```pine
ATR_SL_Mult = 1.8  // 指数波动大，止损可放宽
MinVolumeRatio = 1.2  // 指数流动性好，可降低
```

**外汇 (EURUSD, GBPUSD)**:
```pine
ATR_SL_Mult = 1.4  // 外汇波动小，止损可收紧
TrailStartRatio = 80%  // 外汇趋势弱，可早点锁定
```
