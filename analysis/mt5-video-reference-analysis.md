### 3. 操作流程 ✅

**从OCR分析推测的操作步骤：**

1. **启动MT5**
   - 双击图标
   - 等待加载

2. **打开策略测试器**
   - 快捷键：Ctrl+R
   - 或菜单：查看 → 策略测试器

3. **选择EA**
   - 在策略测试器中选择EA
   - 27箱模型tradingview版 或 nengliang箱模型

4. **设置参数**
   - 交易品种：XAUUSD / US500 / USTEC / IT40
   - 时间周期：H1 / H4
   - 测试模式：每个报价
   - 时间范围：开始日期 → 结束日期

5. **开始回测**
   - 点击"开始"按钮
   - 观察图表回放

6. **查看结果**
   - 切换到"结果"标签
   - 查看统计数据
   - 查看盈亏曲线

7. **导出报告**
   - 右键 → 保存报告
