## MT5 策略测试器操作步骤（推测）

### 步骤1：启动 MT5
```
操作：双击 MT5 图标
结果：MT5 主界面打开
```

### 步骤2：打开策略测试器
```
操作：按 Ctrl+R 或 菜单 → 查看 → 策略测试器
结果：策略测试器窗口出现在底部
```

### 步骤3：选择 EA
```
操作：在策略测试器中选择 EA
选项：
  - 27箱模型tradingview版
  - nengliang箱模型tradingview版本
```

### 步骤4：设置测试参数
```
参数设置：
  - 交易品种：XAUUSD / US500 / USTEC / IT40
  - 时间周期：H1 / M15 / M5
  - 测试模式：每个报价 / 仅开盘价
  - 时间范围：开始日期 → 结束日期
  - 初始资金：1,000,000
  - 杠杆：1:100 / 1:500
```

### 步骤5：设置 EA 参数
```
EA 参数（基于 tradingview-config.txt）：
  - EnablePC: true
  - PC_Lots: 0.01
  - BaseRisk: 1.0%
  - EnableATRStopLoss: true
  - ATR_Period: 14
  - ATR_SL_Mult: 2.0
  - MartinMax: 2
  - MartinMult: 1.4
  - TrailStartRatio: 65%
  - TrailStepRatio: 32%
```

### 步骤6：开始回测
```
操作：点击"开始"按钮
结果：
  - 图表开始回放
  - 交易记录出现
  - 统计数据更新
```

### 步骤7：查看结果
```
查看内容：
  - 图表标签：查看交易点位
  - 结果标签：查看统计数据
  - 报告标签：查看详细报告
  - 图形标签：查看盈亏曲线
```

### 步骤8：导出结果
```
操作：右键 → 保存报告
格式：HTML / CSV
位置：MQL5\Files\
```
